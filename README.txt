A simple snake game that demonstrates BFS, DFS and A* search in Java


====================================================
How To Run:
====================================================
This is a simple text-based game that works by command arguments.
Once arguments are accepted it will run for the time given.
(if not accepted will use default arguments of 10x10 grid with snake length 4 and 10 obstacles using BFS for 2 seconds)


Arguments are easy to enter through eclipse
If you're not familiar with Eclipse follow these steps to enter your arguments:
1. click on Run
2. click on run configurations
3. click on Arguments (should be second tab between Main and JRE tabs)
4. enter arguments



How to enter the arguments:

-Gx = maximum grid of size x
-Gy = maximum grid of size y
-Ss = Snake length
-o = How many obstacles you would like placed (obstacles are placed only once throughout the game)
-st = Type of search you want the snake to have
	1 = BFS
	2 = DFS
	3 = A*
-h = Heuristics. If you're not using A* then -h is ignored.
	1 - Heuristic 1
	2 - Heuristic 2
	3 - Average of  both Heuristic 1 and 2

-s = Simulation in seconds 
 	i.e. entering 6 = 6 seconds.


-------------------
EXAMPLE ARGUMENTS:|
-------------------
Here are some sample ones:

10x10 grid with snake of length 4 and 5 obstacles using BFS for 6 seconds
-Gx 10 -Gy 10 -Ss 4 -o 5 -st 1 -h 0 -s 6

10x10 grid with snake of length 4 and 5 obstacles using DFS for 3 seconds
-Gx 10 -Gy 10 -Ss 4 -o 5 -st 2 -h 0 -s 3


10x10 grid with snake of length 4 and 5 obstacles using A* with heuristic1 for 3 seconds
-Gx 10 -Gy 10 -Ss 4 -o 5 -st 3 -h 1 -s 3
